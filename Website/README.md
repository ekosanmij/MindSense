# MindSense Website

This directory contains the rebuilt marketing website for MindSense AI.

## Files

- `index.html`: complete page structure and product messaging
- `styles.css`: full visual system, responsive layout, and motion styles
- `app.js`: navigation, reveal animations, counters, loop highlight, and tab switching

## Local Preview

```bash
python3 -m http.server 8080 --directory Website
```

Then open `http://127.0.0.1:8080/index.html`.
